# Database Labs

Лабораторні роботи з дисципліни "Бази даних".

## Використані технології

* PostgreSQL
* Prisma ORM
* Node.js
* DataGrip
* Visual Studio Code

---

# Структура репозиторію

## Lab 1 — ER Diagram

Створення ER-діаграми бази даних інтернет-магазину.

## Lab 2 — DDL

Створення структури бази даних за допомогою SQL DDL команд:

* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* CHECK
* UNIQUE

## Lab 3 — OLTP

Виконання OLTP-запитів:

* SELECT
* INSERT
* UPDATE
* DELETE
* JOIN
* GROUP BY

## Lab 4 — OLAP

Виконання аналітичних OLAP-запитів:

* SUM
* AVG
* COUNT
* GROUP BY
* аналітика по категоріях товарів

## Lab 5 — Normalization

Нормалізація бази даних до третьої нормальної форми (3NF):

* функціональні залежності
* 1NF
* 2NF
* 3NF
* фінальна структура таблиць

## Lab 6 — Migrations

Використання Prisma ORM та Prisma Migrate:

* schema.prisma
* migrations
* migration SQL files
* migration-notes.md

# Примітка

Для деяких лабораторних робіт окремі SQL-файли не були збережені під час початкового виконання роботи.
У репозиторії представлені фінальні версії SQL-схем та migration-файлів, які відображають підсумкову структуру бази даних.

Усі етапи виконання лабораторних робіт, SQL-запити та результати їх виконання можна переглянути у PDF/DOCX звітах та на скріншотах, що додані до відповідних папок лабораторних робіт.
